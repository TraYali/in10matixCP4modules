# Laboratory on modules
## Examples
The examples provided are all called in `main.cpp` and are separated according to `CMakeLists.txt`.

## Tasks
The tasks are in the `README.md` in the `tasks/` folder.

## CMake
It was used in attempts to streamline the compilation.
The CMake files are provied as a possible starting point for compilation.

# Possible problems
It's advised to use **Visual Studio** for compiling or *MSVC* if you're using **Visual Studio Code**, but it worked on **clang** on our end with the provided files.
